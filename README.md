# K-Means
K-Means Clustering on a Dense Network Topology
Exploring Energy Distribution Across 600+ Interconnected Nodes

What This Project Is About

Networks can be messy. When you have over 600 nodes scattered across a space, each carrying its own energy level, making sense of the structure isn't something you can do just by looking at it. This project tackles that problem by applying K-Means clustering to uncover hidden groupings within a dense network topology.
Each node in the network has three properties: its position in space (x and y coordinates) and an energy level. The clustering algorithm groups nodes based on these properties, and the results reveal patterns in how energy is distributed across the network — patterns that wouldn't be obvious otherwise.

Goals

Apply K-Means clustering using a provided KMeans() implementation
Make sense of a complex, data-heavy network that resists visual interpretation
Use Python to extract and communicate meaningful insights
Build 2D and 3D visualizations of the clustering results
Pinpoint critical nodes, particularly those with dangerously low energy levels


Tools and Libraries

Python — core language
NumPy — numerical operations
Pandas — data handling
Matplotlib — visualization
Seaborn — optional, for enhanced plots
